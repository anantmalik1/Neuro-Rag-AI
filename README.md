# ⚙️ INSTALLATION

```bash
git clone https://github.com/anantmalik1/Neuro-Rag-AI.git

cd Neuro-Rag-AI

npm install

npm run dev
```

---

# 🌐 ENVIRONMENT VARIABLES

Create `.env` file:

```env
OPENAI_API_KEY=your_api_key

MONGODB_URI=your_database_url

PINECONE_API_KEY=your_pinecone_key
```
